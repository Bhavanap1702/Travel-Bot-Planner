
# TravelBot – Travel Bot Planner

## Overview
TravelBot is a chatbot-based travel planning application that helps users plan their trips easily. It provides information about different destinations, gives travel suggestions, and helps users with basic itinerary planning through a simple conversational interface.

The system uses Natural Language Processing to understand user queries and provide helpful travel-related responses.

## Features
- Chatbot-based travel assistant
- Destination suggestions
- Travel recommendations
- Basic itinerary planning
- Simple and interactive user interface

## Technologies Used
- Python
- Natural Language Processing (NLP)
- Large Language Models (LLM)
- Retrieval Augmented Generation (RAG)
- LangChain
- Hugging Face Transformers
